# A Field Square Root

[Fermat's little theorem](https://en.wikipedia.org/wiki/Fermat%27s_little_theorem):

- $a^p ≡ a \ (\text{mod} \ p)$,
- $a^{p-1} ≡ 1 \ (\text{mod} \ p)$,
- $a^{p-2} = a^{-1} \ (\text{mod} \ p)$.

$\forall a \in \{1, 2, \ldots, p-1\}$.

## Equation

$x^2 ≡ c \ (\text{mod} \ p)$, and $x$ is unknown.
