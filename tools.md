# Tools

## Programming Languages

- [Rust](https://www.rust-lang.org/tools/install).
  - Tarpaulin: `cargo install cargo-tarpaulin`. Run `cargo tarpaulin --out Html`. It will create a `tarpaulin-report.html` file.
